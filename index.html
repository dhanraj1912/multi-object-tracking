How to Use

Step 1: Prepare Your Video

Place your input video file as input.mp4 in the project directory

Step 2: Run the Tracker

python main.py

Step 3: View Results

Tracked output video is saved as output.mp4

How It Works

  1. Detection: YOLOv8 detects all persons in each frame
  2. Feature Extraction: DeepSORT extracts appearance features for each detection
  3. Matching: Associates detections across frames using appearance + motion
  4. ID Assignment: Maintains consistent IDs even through occlusion
  5. Visualization: Annotates video with bounding boxes and ID numbers
